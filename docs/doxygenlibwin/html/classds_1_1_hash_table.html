Detailed Description

template<typename K, typename V>
class ds::HashTable< K, V >

Kapalı adreslemeli hash tablosu implementasyonu.

Template Parameters
KAnahtar türü
VDeğer türü

Bu sınıf, kapalı adresleme (closed addressing) tekniği kullanarak hash tablosu implementasyonu sağlar. Linear probing ile çakışma çözümü yapar.
